# Vibe Patent Core Setup

## Quick Start
1. Add the marketplace: `/plugin marketplace add patrick-ip-law/vibe-patent-marketplace`
2. Install core plugin: `/plugin install vibe-patent-core`
3. Get your free/paid key at https://vibepatent.ai
4. Run `/vibe-patent-core:check-account` with your key to verify credits.

## MCP Server Options
- **Recommended (Hosted)**: Uses the production Streamable HTTP endpoint.
- **Advanced (Local)**: Run via `.mcp.json` with `SUPABASE_URL` and `SUPABASE_ANON_KEY`.
